# Spring Boot Microservices Project

## 🚀 Services

* User Service
* Hotel Service
* Rating Service
* API Gateway
* Eureka Server
* Config Server

## 🧩 Tech Stack

* Java
* Spring Boot
* Spring Cloud
* MySQL
* Mongo DB
* Resilience4j

## 🔥 Features

* Microservices Architecture
* Circuit Breaker
* Service Discovery
* Centralized Config

## ▶️ How to Run

1. Start Config Server
2. Start Eureka Server
3. Start Services
4. Start API Gateway

## 📬 Postman Collection

You can test all APIs using the Postman collection:

📁 Location: `/postman/microservices-postman-collection.json`

### ▶️ Steps to Use:
1. Open Postman
2. Click **Import**
3. Upload the JSON file
4. Run APIs
