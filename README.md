# Spring Boot Microservices Project

## 🚀 Services

* User Service
* Hotel Service
* Rating Service
* API Gateway
* Eureka Server
* Config Server

## 🧩 Tech Stack

* Java
* Spring Boot
* Spring Cloud
* MySQL
* Resilience4j

## 🔥 Features

* Microservices Architecture
* Circuit Breaker
* Service Discovery
* Centralized Config

## ▶️ How to Run

1. Start Config Server
2. Start Eureka Server
3. Start Services
4. Start API Gateway
